# DegeneracyCountingQuantumAlgorithm
This is dat aand code for paper Degeneracy Counting Quantum Algorithm using Decoherence

# Abstract

Counting the global optima of a classical optimization problem, or determining the ground-state degeneracy $g_P$ of a corresponding problem Hamiltonian, is a #P-hard task for which existing quantum approaches are constrained by the cost of full-register measurement. We develop and investigate the Canonical Thermal Pure Quantum (CTPQ) state-based Degeneracy Counting algorithm, CTPQsd#, a decoherence-based algorithm that determines $g_P$ by measuring only a small probe system, without attempting to find individual degenerate states. The method exploits a perturbative relation between $g_P$ and the decoherence measure σS of the probe system S when the probe and problem are together in a CTPQ state. We provide the first numerical demonstration that this relation can be used to extract $g_P$, and we show that it holds for a diagonal random-energy Hamiltonian whose energy levels are drawn from a uniform random distribution, serving as a maximally unstructured, rigorous testbed for the broader class of diagonal Hamiltonians used to encode arbitrary classical binary optimization problems. Using classical simulations, we evaluate the robustness of the algorithm to variations in the temperatures used to prepare the S+P CTPQ state, the Hamiltonian energy scale R, the problem system size $N_P$, and the problem degeneracy $g_P$. We further identify a two-threshold regime in which an intermediate temperature window counts near-degenerate minima within a user-defined energy tolerance, thereby extending the method from exact to approximate degeneracy. By confining measurement to the probe, the protocol enables replacing tomography over the full problem Hilbert space with tomography over a fixed, few-qubit system, offering a potentially scalable route to implementing the algorithm on future quantum hardware.
