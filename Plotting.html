Plot Creator

1 · Data

Paste raw text (Excel format)
Click any cell to edit. Paste a range from Excel into any cell — the table will auto-expand. × next to a row or a series header deletes it; + at the bottom-left adds a row, + at the header right adds a series. Series names are editable in the header row.
Error columns: add a new series with +, then rename it with the suffix _err (symmetric ±), or _err+ and _err- (asymmetric upper / lower). On blur the column is auto-linked to its base series (e.g. g_P=1_err+ → g_P=1) and shows up as the +err of g_P=1 tag.

2 · Axes

Y title & plot title support LaTeX (wrap in $…$, e.g. $\sqrt{\langle \sigma^2 \rangle}$) and HTML (e.g. &sigma;<sup>2</sup>). Note: LaTeX in PNG/SVG export can be unreliable — use HTML if export quality matters.

3 · Series

Paste data first.

4 · Interaction

When on, you'll see halo rings around each data point — drag a ring vertically to bend the guide curve through that location. Your original data is never modified; guides are hidden in PNG/SVG export. Hold Shift to snap to 0.01. The legend is always draggable.

5 · Export

Plot style options
Text size affects axis titles, tick labels, and legend (unless Legend text size is set). The plot title is automatically 2 px larger.
Tip: copy any contiguous range from Excel (header row + data) and paste it into any cell of the data table — the table will auto-expand to fit the block.
